什么是Verdaccio ​

Verdaccio 是一个Node.js创建的轻量级 npm 私有仓库框架

Verdaccio能做什么 ​

  • 发布私有包，不被团队外部的人使用
  • 通过proxy下载私有仓库没有的包
  • 缓存私有仓库没有的包，方便下次安装更快
  • 设置私有仓库的访问权限
  • 使用Docker、AWS等部署方式

使用背景 ​

  • 离线化开发
  • 本地工程安装依赖包比较繁琐
  • 版本管理存在缺陷

Verdaccio的使用 ​

  1. 安装
md
npm install --global verdaccio
  1. 启动 Verdaccio
md
// 在命令行中，输入 verdaccio 即可
verdaccio
  1. 默认访问地址
md
http://localhost:4873/
  1. Verdaccio配置

     配置详见 config.yaml

指定 npm registry ​

md
npm set registry http://localhost:4873/
// 或
NPM_CONFIG_REGISTRY=http://localhost:4873 npm install

Publishing ​

  1. 创建用户并登录
md
npm adduser --registry http://localhost:4873
  1. 推送npm包到服务上
md
npm publish --registry http://localhost:4873

常见问题 ​

  • 上传到 verdaccio 库的包存放在什么地方？

答：

默认公共配置 C:\Users\hp.config\verdaccio\storage，config文件也在此处

  • 如何将 verdaccio 连同拥有的npm包一起copy到其他环境或主机

答：

将verdaccio包及其配置与库文件 C:\Users\hp\.config\verdaccio\* 一起带走即可
